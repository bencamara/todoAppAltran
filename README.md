## todoApp Altran Test

## Installation

install des dependances.
```
npm install or yarn install
```

## Utilisation
``` npm start ```
 Le port utilisée est 4300.l'utilisateur peut le modifier à souhait via la commande   ``` ng serve --port numero_du_port ```
